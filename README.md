# 💎 Fynz - Personal Finance Manager

Fynz es una aplicación diseñada para simplificar el seguimiento de tus gastos, optimizar tus ahorros y ofrecerte una visión clara de tu salud financiera en tiempo real.

Este repositorio contiene el **Frontend** de la plataforma.

## 🌟 Visión General
Construida con un enfoque en la velocidad y la experiencia de usuario (UX), Fynz permite a los usuarios:
- Monitorear saldos mensuales.
- Categorizar gastos de forma automática.
- Establecer metas de ahorro mediante "Bolsillos".
- Acceder a un panel administrativo centralizado.
